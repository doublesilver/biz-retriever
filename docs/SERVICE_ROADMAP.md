# 서비스 로드맵 (SERVICE_ROADMAP.md)

## Phase 1: 데이터 기초 (Data Infrastructure)
- 공공 API 및 스크래핑 체계 구축
- HWP/PDF 독해 엔진 완성
- 24시간 자동화 수집 검증

## Phase 2: 프로필 자동화 (Smart Profile)
- 사업자 등록증 및 실적 증명서 기반 AI 프로필 생성
- 수동 입력 최소화 UX

## Phase 3: 정밀 매칭 (Zero-Error Matching)
- Hard Match 엔진 (지역, 제한사항, 면허)
- 매칭 논거(Logic) 생성기

## Phase 4: 대시보드 & 통계 (Insight Dashboard)
- 수집 현황 및 매칭 통계 시각화
- 공고 요약 기능

## Phase 5: 신뢰 QA (Trust & Compliance)
- 법적 리스크 전면 검토
- 오탐 제로 테스트

## Phase 6: 프로덕션 런칭 (Production Go-Live)
- Raspberry Pi 최종 최적화
- 안정 운영 가이드 배포

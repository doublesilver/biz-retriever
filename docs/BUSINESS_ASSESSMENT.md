## 💼 비즈니스 모델 분석

### 시장 규모 및 기회

**목표 시장**
- **나라장터 (G2B)**: 연간 거래액 **120조 원** (2021년 기준)
- **타겟 고객**: 중소기업 (SME) 입찰 참여 업체 약 **50만 개**
- **Pain Point**: 매일 1~2시간 수동 입찰 검색 → 연간 500시간 낭비

**시장 성장성**
- ✅ 정부 디지털 전환 정책 (2026년 중소기업 지원 예산 2배 확대)
- ✅ AI 입찰 분석 수요 증가 (2025년 YoY 60% 성장)
- ✅ 재택근무 확산 → 클라우드 기반 협업 도구 수요

### 경쟁사 분석

| 경쟁사 | 가격 | 강점 | 약점 |
|--------|------|------|------|
| **모두입찰** | 27,500원/월 | 시장 선점, 브랜드 인지도 | AI 분석 없음, 단순 알림 |
| **GOBID** | 성과 기반 (%) | No upfront cost | 수수료 높음 (5-10%) |
| **Cliwant** | 크레딧 방식 | 유연한 요금제 | 복잡한 가격 구조 |
| **Biz-Retriever** | 29,000원/월 (Pro) | **AI 분석**, Hard Match | **사용자 0명**, 검증 없음 |

**경쟁 우위**
- ✅ Google Gemini AI 기반 자동 분석 (경쟁사 미제공)
- ✅ Hard Match 엔진 (지역/면허/실적 자동 검증)
- ✅ 무료 API (Gemini 1,500 req/일) → 비용 경쟁력

**경쟁 열위**
- ⚠️ 브랜드 인지도 0%
- ⚠️ 고객 확보 채널 없음 (마케팅 예산 0원)
- ⚠️ 네트워크 효과 없음 (신규 진입자 불리)

### 단위 경제성 (Unit Economics)

**가정 기반 계산** (실제 검증 필요)

| 지표 | 값 | 근거 |
|------|-----|------|
| **ARPU** (월간)| 29,000원 | Pro 플랜 기준 |
| **Churn Rate** | 10%/월 | 가정 (업계 평균 5-15%) |
| **평균 사용 기간** | 10개월 | 1 / Churn Rate |
| **LTV** | 290,000원 | ARPU × 10개월 |
| **CAC** | 75,000원 | 가정 (온라인 광고 CPA) |
| **LTV:CAC 비율** | **3.87:1** | ✅ 건강 (목표 >3.0) |

**손익분기점 (Break-Even Point)**
- 월 고정 비용: 약 50만 원 (서버 5만 + 개발자 인건비 환산 45만)
- 필요 고객 수: **18명** (50만 / 29,000원)
- **검증 필요**: 실제 고객 0명, CAC 추정치 불확실

### 가격 전략 검증 상태

**Freemium 모델**
- **Free**: 일일 맞춤 공고 3건
- **Pro** (29,000원/월): 무제한 + AI 분석 + 승률 예측

**검증 안 된 가정**
- ❌ 무료 → 유료 전환율 (가정 5%, 실제 0%)
- ❌ 29,000원 지불 의향 (경쟁사 대비 +5.5% 비쌈)
- ❌ 3건 제한이 충분한 teaser인지 불명
- ❌ AI 분석 가치에 대한 사용자 인식 미검증

## ⚠️ 위험 요소 및 완화 전략

### 1. 시장 검증 실패 위험 (확률: 70%)

**위험**: 실제 사용자가 29,000원 지불 의향 없을 수 있음

**영향**: 매출 0원 지속, 프로젝트 폐기

**완화 전략**:
- **즉시**: 무료 베타 사용자 50명 모집 (네이버 카페, 입찰 커뮤니티)
- **2주**: 사용 패턴 분석 및 유료 전환 설문
- **1개월**: 실제 전환율 측정 (목표 >3%)

### 2. 경쟁사 AI 기능 추가 위험 (확률: 60%)

**위험**: 모두입찰 등 기존 업체가 AI 기능 추가 시 차별화 상실

**영향**: 경쟁 우위 소멸, 시장 진입 실패

**완화 전략**:
- **즉시**: Hard Match 엔진 고도화 (타사 불가능한 기능)
- **3개월**: 승률 예측 정확도 95% 달성 (데이터 축적 필요)
- **6개월**: 입찰 컨설팅 서비스로 확장 (SaaS → Professional Service)

### 3. 라즈베리파이 인프라 장애 위험 (확률: 90%)

**위험**: SD 카드 고장 시 데이터 전체 유실

**영향**: 서비스 중단, 사용자 신뢰 상실

**완화 전략**:
- **즉시 (Wave 1)**: 자동 백업 시스템 구축 (매일 3AM)
- **1주**: 외장 SSD 구매 (2-4만 원) 및 PostgreSQL 마이그레이션
- **2주**: 클라우드 백업 (AWS S3 Glacier - 월 1,000원)

### 4. 고객 확보 채널 부재 위험 (확률: 80%)

**위험**: 마케팅 예산 0원, 유입 경로 없음

**영향**: 사용자 증가 정체, 손익분기점 미달

**완화 전략**:
- **즉시**: 네이버 블로그 SEO 최적화 (무료)
- **1개월**: 입찰 관련 카페/커뮤니티 활동 (수작업 마케팅)
- **3개월**: 파트너십 (세무사 사무소, 건설업 협회)
- **6개월**: Google Ads (월 30만 원 예산)

### 5. 법적 리스크 (확률: 30%)

**위험**: 공공데이터 활용 약관 위반, 개인정보보호법 저촉

**영향**: 서비스 중단 명령, 법적 제재

**완화 전략**:
- **즉시**: G2B API 이용 약관 재확인
- **1주**: HTTPS 적용 (개인정보보호법 PIPA 준수)
- **1개월**: 개인정보처리방침 게시
- **3개월**: 법률 자문 (변호사 검토 50만 원)
